# Утилита NTP Time
Получает точное время с NTP-сервера (`time.google.com`).

## Установка
Компиляция утилиты:
```sh
go build -o ntptime ./ntptime
```

**Пример использования:**
```sh
./ntptime
```
Вывод:
```
Точное время: 2024-02-24 12:34:56 +0000 UTC
```
