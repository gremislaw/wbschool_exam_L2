# Утилита распаковки строки
Программа распаковывает строки вида `a4bc2d5e`, заменяя цифры на повторяющиеся символы.

## Установка
```sh
go build -o unpack ./unpack
```

**Пример использования:**
```sh
./unpack "a4bc2d5e"
```
Вывод:
```
aaaabccddddde
```
